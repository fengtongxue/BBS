# 功能
- 用户管理
    - 用户注册
    - 用户登录
    - 添加头像
    - 个人签名，用户名更改
    - 通过重置密码
- 话题管理
    - 话题创建
    - 话题回复
    - 语法高亮
    - 删除话题自动删除所有相关回复
    - 不同板块归类管理 
- 站内信
    - 邮件通知站内信
    - 邮件通知@ 用户
